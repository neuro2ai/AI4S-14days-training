4. 为什么要定义forward(def forward(self, x):)

5. 为什么需要ReLU做非线性变换，linear-linear本质还是线性变换有什么不好，如果用其他函数做非线性变换，目的和结果上有什么不同，怎么根据目的不同选择不同的函数进行非线性变换？  

多个线性仍等于一个单层线性  

| 函数 | 特点 | 什么时候用 |
| :--- | :--- | :--- |
| ReLU | 简单、快、稳定 | 默认选择（90%情况） |

| Tanh | 输出 -1-1 | 有时用于对称数据 |

# 3. 今天最重要的3个概念

1. super().__init__(self)调用父类（nn.Module)的初始化  

2. x = self.relu(x) 的意思是：“把数据 x 丢进 relu 机器里加工，加工完产生的新结果，再重新命名为 x，称之为“覆盖更新”。ReLU 的数学定义：f(x) = max(0, x)。  

3. def forward(self, x):定义了数据从输入到输出的计算路径，每一层对数据做变换，最终得到新的表示。
